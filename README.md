# Product Dataset Exploratory Data Analysis (EDA)

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a product dataset using Python and Pandas in Google Colab.

## Objectives

- Explore the dataset
- Check dataset dimensions
- Identify missing values
- Detect duplicate records
- Analyze product prices
- Analyze customer ratings
- Analyze discounts
- Study product categories and stock status

## Technologies Used

- Python
- Pandas
- NumPy
- Google Colab

## Dataset Information

The dataset contains the following columns:

- Category
- Price
- Rating
- Stock
- Discount

## Analysis Performed

- Dataset shape
- Column names
- Data types
- Missing value analysis
- Duplicate record analysis
- Statistical summary
- Minimum, Maximum and Mean calculations
- Category-wise analysis
- Stock status analysis
